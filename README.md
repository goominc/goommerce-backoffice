Goommerce Back Office
=====================

We use '[Metronic](http://themeforest.net/item/metronic-responsive-admin-dashboard-template/4021469)' as a template. You should buy it before use.

For testing,

> $ cd backoffice
>
> $ gulp --dev
>
> $ cd ..
>
> $ bower install
>
> $ npm install
>
> $ node server
